# Student Score Analyzer

This is a simple Python project that analyzes students' scores.

## Description

The program allows the user to enter student scores and then calculates:

- Largest score
- Lowest score
- Average score
- Total number of students
- Grade distribution (A, B, C, D, F)

The user can enter scores until typing **done**.

## Features

- Input validation
- Score range checking (0 - 100)
- Average calculation
- Grade classification
- Largest and lowest score detection
- Error handling

## How to Run

1. Open terminal
2. Go to project folder
3. Run the program

```bash
python students_system.py
